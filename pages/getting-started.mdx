## Usage

Once you installed c3pm, you can start using it.
Here are some basic commands you will need.

### Start your project
```shell
ctpm init
```

### Add a package
```shell
ctpm add <package>
```


### Building your project
```shell
ctpm build
```

### Publishing your project

```shell
ctpm publish
```

<br />

You can find a more complete list of the available commands [here](https://github.com/c3pm-labs/specs/tree/main/cli) !
